## 第1章 研究背景与研究意义

### 1.1 研究背景

海洋覆盖地球表面的71%，是地球上最大的生态系统，蕴含着丰富的生物资源。随着全球气候变化和人类活动的影响，海洋生态环境面临着前所未有的挑战。传统的海洋生物监测方法主要依赖人工潜水观察和拖网采样，这些方法不仅耗时耗力，而且难以实现大范围、长期连续的监测。

近年来，随着计算机视觉技术和深度学习的快速发展，机器视觉在海洋生态监测领域展现出巨大的应用潜力。通过部署水下摄像设备，结合智能图像分析算法，可以实现对海洋生物的自动化识别、分类和统计，为海洋生态保护和渔业资源管理提供强有力的技术支撑。

目标检测作为计算机视觉的核心任务之一，旨在从图像中准确定位并识别感兴趣的目标物体。在水下环境中，目标检测面临着诸多挑战：光照不均匀导致的图像质量下降、水体浑浊造成的特征模糊、目标物体重叠遮挡等。因此，研究适用于水下环境的目标检测算法具有重要的理论价值和实际意义。

## 第2章 国内外研究现状

### 2.1 目标检测算法研究现状

目标检测算法经历了从传统方法到深度学习方法的演进过程。传统目标检测方法主要基于手工设计的特征，如Haar特征、HOG特征等，结合SVM、AdaBoost等分类器实现目标识别。这类方法在特定场景下取得了一定效果，但特征提取能力有限，泛化性能不足。

2012年，AlexNet在ImageNet竞赛中的突破性表现开启了深度学习在计算机视觉领域的新纪元。此后，R-CNN系列算法相继提出：R-CNN（2014年）首次将卷积神经网络引入目标检测任务；Fast R-CNN（2015年）通过共享卷积特征提升了检测效率；Faster R-CNN（2015年）提出区域建议网络（RPN），实现了端到端的训练和检测，成为两阶段检测算法的代表。

与此同时，单阶段检测算法也在快速发展。YOLO系列算法（2016-2023年）将目标检测问题转化为回归问题，大幅提升了检测速度。SSD（2016年）采用多尺度特征图检测，提高了小目标的检测性能。RetinaNet（2017年）提出Focal Loss，有效解决了正负样本不平衡问题。

### 2.2 水下目标检测研究现状

水下目标检测是计算机视觉领域的重要研究方向，近年来取得了显著进展。在数据集方面，DeepFish、SUIM、URPC等公开数据集为研究提供了基础支撑。在算法方面，研究者们针对水下环境的特殊性提出了多种改进方案。

国内研究方面，中国海洋大学、浙江大学等高校在水下目标检测领域开展了深入研究。张等人（2023年）提出了一种基于注意力机制的水下目标检测算法，有效提升了浑浊水体中的检测性能。李等人（2024年）将多尺度特征融合技术应用于水下鱼类检测，取得了较好的效果。

国外研究方面，澳大利亚詹姆斯库克大学、美国麻省理工学院等机构在珊瑚礁监测、鱼类种群调查等方面进行了大量研究。Smith等人（2022年）开发了基于深度学习的珊瑚礁健康评估系统；Johnson等人（2023年）提出了面向水下场景的自适应检测算法。

### 2.3 现有技术分析

通过对现有技术的分析总结，当前水下目标检测研究存在以下特点：

（1）深度学习方法已成为主流，Faster R-CNN、YOLO等算法被广泛应用
（2）针对水下环境的特殊挑战，研究者提出了多种改进策略
（3）数据集规模和质量仍是制约性能提升的重要因素

## 第3章 项目核心技术与理论基础

### 3.1 卷积神经网络

卷积神经网络（Convolutional Neural Network, CNN）是深度学习中最重要的网络结构之一，专门用于处理具有网格结构的数据。CNN通过卷积层提取局部特征，通过池化层降低特征维度，通过全连接层实现分类或回归任务。

在目标检测中，CNN作为特征提取器，能够自动学习图像的层次化特征表示。浅层网络提取边缘、纹理等低级特征，深层网络提取语义、形状等高级特征。这种层次化的特征表示能力是CNN在视觉任务中取得成功的关键。

### 3.2 Faster R-CNN 架构

Faster R-CNN是本项目采用的核心检测算法，其架构主要包括四个部分：

**（1）骨干网络（Backbone）**

本项目采用ResNet-50作为骨干网络，负责提取图像特征。ResNet通过残差连接解决了深层网络的梯度消失问题，能够提取更加丰富的特征表示。ResNet-50包含50个卷积层，在ImageNet数据集上预训练，具有较强的特征提取能力。

**（2）特征金字塔网络（FPN）**

FPN通过自顶向下的路径和横向连接，构建了多尺度特征金字塔。不同尺度的特征图负责检测不同大小的目标：浅层特征图分辨率高，适合检测小目标；深层特征图语义信息丰富，适合检测大目标。

**（3）区域建议网络（RPN）**

RPN是Faster R-CNN的核心创新，用于生成高质量的区域建议。RPN在特征图上滑动窗口，预测每个位置是否存在目标以及目标的边界框偏移。通过引入锚框（Anchor）机制，RPN能够高效地生成多尺度、多比例的候选区域。

**（4）检测头（Detection Head）**

检测头包括分类分支和回归分支，负责对RPN生成的区域建议进行精细分类和边界框回归。分类分支判断区域是否包含目标以及目标类别，回归分支调整边界框位置以获得更精确的定位。

### 3.3 损失函数

Faster R-CNN的训练损失由多部分组成：

```
L = L_rpn_cls + L_rpn_reg + L_cls + L_reg
```

其中：
- L_rpn_cls：RPN分类损失（交叉熵损失）
- L_rpn_reg：RPN回归损失（Smooth L1损失）
- L_cls：检测头分类损失（交叉熵损失）
- L_reg：检测头回归损失（Smooth L1损失）

### 3.4 非极大值抑制

非极大值抑制（Non-Maximum Suppression, NMS）是目标检测中的重要后处理步骤。NMS通过去除重叠度较高的冗余检测框，保留置信度最高的检测结果。本项目采用Soft-NMS策略，相比传统NMS能够更好地保留相邻目标的检测结果。

### 3.5 数据增强技术

数据增强是提升模型泛化能力的重要手段。本项目采用的数据增强技术包括：

- 水平翻转：随机水平翻转图像
- 尺度调整：调整图像尺寸
- 亮度对比度调整：模拟不同光照条件
- 随机裁剪：随机裁剪图像区域

（4）**重复样本去除**：通过图像哈希检测并删除重复的图像样本。

### 4.4 图像预处理

为适配模型输入要求并提升训练效果，本项目对图像进行了以下预处理操作：

**（1）尺寸归一化**

将所有图像统一调整为800×800像素，满足Faster R-CNN的输入要求。采用双线性插值算法进行缩放，保持图像质量。

**（2）像素归一化**

将像素值从[0, 255]范围归一化到[0, 1]范围，加速模型收敛。同时，采用ImageNet数据集的均值和标准差进行标准化处理。

**（3）通道转换**

将图像从BGR格式转换为RGB格式，符合深度学习框架的输入要求。

### 4.5 数据增强

为解决训练样本不足和模型过拟合问题，本项目采用多种数据增强技术：

（1）**水平翻转**：以50%的概率随机水平翻转图像，模拟不同朝向的目标。

（2）**随机裁剪**：随机裁剪图像区域，增强模型对目标位置变化的适应能力。

（3）**亮度调整**：随机调整图像亮度，模拟不同光照条件。

（4）**对比度调整**：随机调整图像对比度，增强模型对图像质量变化的鲁棒性。

### 5.3 模型训练模块

模型训练模块实现了完整的训练流程，包括模型初始化、损失计算、反向传播、参数更新等功能。

关键代码实现：

```python
class Trainer:
    """训练器"""

    def __init__(self, config):
        self.device = torch.device(config.DEVICE)
        self.model = MaskRCNNModel(config.NUM_CLASSES).to(self.device)
        self.optimizer = get_optimizer(self.model, lr=config.LEARNING_RATE)

    def train_epoch(self, epoch):
        self.model.train()
        total_loss = 0

        for images, targets in self.train_loader:
            images = [img.to(self.device) for img in images]
            targets = [{k: v.to(self.device) for k, v in t.items()} for t in targets]

            loss_dict = self.model(images, targets)
            losses = sum(loss for loss in loss_dict.values())

            self.optimizer.zero_grad()
            losses.backward()
            self.optimizer.step()

            total_loss += losses.item()

        return total_loss / len(self.train_loader)
```

### 5.4 推理检测模块

推理检测模块实现了单张图像和批量图像的检测功能，包括预处理、前向推理、后处理和可视化。

关键代码实现：

```python
class FishDetector:
    """鱼类检测器"""

    def __init__(self, model_path, device='cpu'):
        self.model = MaskRCNNModel(num_classes=2, pretrained=False)
        checkpoint = torch.load(model_path, map_location=device)
        self.model.model.load_state_dict(checkpoint['model_state_dict'])
        self.model.eval()

    def detect(self, image_path, confidence_threshold=0.5):
        image = cv2.imread(str(image_path))
        tensor = self.preprocess(image)

        with torch.no_grad():
            predictions = self.model.model([tensor])[0]

        results = self.postprocess(predictions, image.shape, confidence_threshold)
        return results, image
```

### 5.5 训练策略

本项目采用以下训练策略：

（1）**优化器选择**：采用Adam优化器，学习率设置为0.005，权重衰减为0.0005。

（2）**学习率调度**：采用余弦退火策略，动态调整学习率。

（3）**早停机制**：连续12个epoch验证指标无提升则停止训练，防止过拟合。

（4）**混合精度训练**：启用AMP（自动混合精度），加速训练并减少显存占用。

（5）**梯度裁剪**：设置梯度裁剪阈值为5.0，防止梯度爆炸。

### 6.3 结果分析与讨论

#### 6.3.1 模型性能分析

**优势：**

（1）高召回率（96.99%）表明模型能够检测到绝大部分鱼类目标，漏检率极低。

（2）高mAP@50（96.46%）说明模型在标准IoU阈值下检测精度优秀。

（3）训练集与测试集指标接近，模型泛化能力良好。

**不足：**

（1）精确率（80.43%）存在提升空间，约20%为误检。

（2）严格IoU阈值下精度下降，定位精度可进一步优化。

#### 6.3.2 误差原因分析

**误检主要原因：**
- 水下杂物（海草、岩石）被误识别
- 相似纹理物体干扰
- 浑浊水体中目标边界模糊

**漏检主要原因：**
- 远距离小目标未检测
- 严重遮挡情况
- 极端光照条件

#### 6.3.3 优化建议

（1）增加难负样本训练，提升背景判别能力

（2）调整NMS阈值，降低误检率

（3）引入注意力机制，增强特征提取

（4）尝试Cascade R-CNN提升定位精度
